КАК ОБНОВИТЬ ЧЕРЕЗ GITHUB DESKTOP

1. Распакуйте архив petrov-expert-site-v8.zip.
2. В GitHub Desktop выберите Repository → Show in Explorer.
3. В отдельном окне откройте распакованную папку petrov-expert-site-v8.
4. В ней нажмите Ctrl + A → Ctrl + C.
5. В папке локального репозитория petrov-expert кликните по пустому месту → Ctrl + V.
6. Подтвердите замену файлов и объединение папки assets.
   Старые изображения, которых нет в V8, могут остаться в assets/images: они не используются сайтом и на работу не влияют.
7. Вернитесь в GitHub Desktop.
8. В поле Summary напишите:
   V8: дизайн, фото, кейсы и контакты
9. Нажмите Commit to main.
10. Сверху нажмите Push origin.
11. Подождите 1–2 минуты, затем откройте сайт и сделайте жёсткое обновление Ctrl + F5.
