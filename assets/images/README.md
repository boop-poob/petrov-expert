# Изображения сайта

В публичной версии использованы только реальные фотографии, переданные для сайта.

- `georgiy-on-site.webp` — исходное фото в каске; кадрирование и цветокоррекция.
- `georgiy-profile.webp` — исходный портрет; кадрирование и цветокоррекция.
- `school-novo-peredelkino.webp`, `schelkovo-networks.webp` — реальные фото объектов.
- `worker-*.webp` — архивные фотографии работ на монументе «Рабочий и колхозница».
- `recognition-*.webp` — переданные документы и памятные награды, связанные с проектом.

Обработка: кадрирование, сжатие для веба, лёгкая коррекция контраста и цвета. Генеративные изображения и сгенерированные портреты в сайт не включены.
