# Как поставить реальные фотографии

Текущая версия сайта использует аккуратную инженерную графику, чтобы не выдавать нейросетевые или стоковые изображения за реальные объекты. Перед публикацией лучше заменить её на подтверждённые фотографии с Tilda и из личного архива.

## Имена файлов

Положите изображения в эту папку и замените `src` в нужной HTML-странице.

- `georgiy-hero.webp` — вертикальный портрет для первого экрана, 1200×1500 px или больше.
- `georgiy-about.webp` — горизонтальный или вертикальный кадр для страницы «О эксперте», от 1200 px по ширине.
- `school.webp` — школа в Ново-Переделкино, горизонталь 1600×1000 px.
- `schelkovo.webp` — очистные сооружения / инженерные сети, горизонталь 1600×1000 px.
- `worker.webp` — монумент «Рабочий и колхозница», горизонталь 1600×1000 px.
- `microdistrict.webp` — ЖК «Микрогород в Лесу», горизонталь 1600×1000 px.

## Правило для фотографий

Использовать только те кадры, на которые есть право публикации. У фотографии объекта должна быть понятная подпись: объект, этап и что именно показано. Не использовать сгенерированные изображения как доказательство реализованного проекта.
