ОБНОВЛЕНИЕ САЙТА V6

1. Распакуйте архив petrov-expert-site-v6.zip.
2. Откройте папку, куда GitHub Desktop скачал репозиторий:
   Documents\GitHub\petrov-expert
3. Откройте папку petrov-expert-site-v6 из распакованного архива.
4. Выделите ВЕСЬ ЕЁ СОДЕРЖИМЫЙ НАБОР ФАЙЛОВ И ПАПОК (Ctrl + A → Ctrl + C).
5. Вставьте в папку репозитория petrov-expert (Ctrl + V).
6. На вопрос Windows выберите «Заменить файлы в папке назначения».
   Папку assets разрешите объединить.
7. Откройте GitHub Desktop.
8. В поле Summary напишите:
   V6: новый стиль, проекты, типографика и форма
9. Нажмите Commit to main.
10. Нажмите Push origin.

После Push origin Vercel обновит сайт автоматически — обычно в течение 1–2 минут.
