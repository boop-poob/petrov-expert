ОБНОВЛЕНИЕ ДО V9

1. Распакуйте архив.
2. Откройте папку petrov-expert-site-v9.
3. Скопируйте ВСЁ содержимое папки.
4. Вставьте в локальную папку petrov-expert, которую открывает GitHub Desktop.
5. Выберите «Заменить файлы в папке назначения».
6. В GitHub Desktop: Summary: V9: единый дизайн, фотографии и структура
7. Commit to main → Push origin.
8. Через 1–2 минуты откройте сайт и нажмите Ctrl+F5.

Важно: e-mail не выводится в интерфейсе. Formspree endpoint сохраняется в assets/js/config.js.
